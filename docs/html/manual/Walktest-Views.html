3.1.4.5 Views Commands

It is possible to run in split-screen mode, as if it were a multiplayer game.

fullscreen boolean [Alt-Enter]

Set to fullscreen mode. This only works with some renderers, as it performs an `extension' command.

split_view [Capital S]

Split to 2 views. The screen is divided into a left and right half. You can navigate independantly in both views. The new view is located back at the map start point.

unsplit_view [Capital U]

Go back to single view.

toggle_view [Capital T]

Switch the active view, ie the one you can move in.
